# Volodymyr Rybak - Interactive Resume Terminal

An interactive terminal-style resume landing page with a modern, animated design.

## Features

- **Interactive Terminal**: Type commands to view different sections of the resume
- **Modern Design**: Dark theme with green terminal aesthetics
- **Smooth Animations**: Fade-in, slide animations, and glowing effects
- **Responsive Layout**: Works on desktop and mobile devices
- **Command History**: Use arrow keys to navigate through command history

## Commands

- `show all` - Display entire resume
- `show about` - Display about me section
- `show education` - Display education section
- `show skill` - Display skills section
- `show experience` - Display experience section
- `show help` - Display help message
- `clear` - Clear terminal

## Setup

1. Place your profile photo as `profile.jpg` in the root directory
2. Open `index.html` in a web browser
3. Start typing commands in the terminal!
